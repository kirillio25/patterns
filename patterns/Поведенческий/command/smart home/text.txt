Разработать систему управления умным домом, где можно отправлять команды для управления различными устройствами:

Освещение (включить/выключить свет).
Музыка (включить/выключить музыку).
Температура (увеличить/уменьшить температуру).
Должна быть возможность:

Создавать команды для каждого устройства.
Сохранять историю выполненных команд, чтобы их можно было отменить (undo).
